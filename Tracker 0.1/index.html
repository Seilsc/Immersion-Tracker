Clave de la API de YouTube

Necesaria para calcular la duración automáticamente desde los enlaces de YouTube. Consíguela gratis en Google Cloud Console: crea un proyecto → activa "YouTube Data API v3" → crea una credencial de tipo API key. Se guarda solo en este navegador (localStorage), nunca se envía a ningún sitio salvo a la propia API de YouTube.

Clave de la API de TMDB

Necesaria para buscar series/animes y obtener la duración media por episodio automáticamente. Consíguela gratis en themoviedb.org/settings/api (crea una cuenta → Configuración → API → solicita una "API Key" tipo v3 auth). Se guarda solo en este navegador.

Vídeos de YouTube

01

Pega uno o varios enlaces (uno por línea, o separados por espacios/comas). La duración de cada vídeo se obtiene automáticamente con la API.

Series y animes

02

Escribe el nombre y búscalo en TMDB. Elige el resultado correcto y se guardará con su duración media por episodio. Después solo actualiza el número de episodios vistos y el total se recalcula al instante.

Películas

03

Escribe el título y búscalo en TMDB. Elige el resultado correcto y se añadirá con su duración real.

Todos los datos se guardan localmente en tu navegador.
