API keys para cálculo automático de duraciones, y gestión de idiomas.

Clave de la API de YouTube

Necesaria para calcular la duración automáticamente desde los enlaces de YouTube. Consíguela gratis en Google Cloud Console: crea un proyecto → activa "YouTube Data API v3" → crea una credencial de tipo API key. Se guarda solo en este navegador, nunca se envía a ningún sitio salvo a la propia API de YouTube.

Clave de la API de TMDB

Necesaria para buscar series/animes y películas y obtener la duración automáticamente. Consíguela gratis en themoviedb.org/settings/api (crea una cuenta → Configuración → API → solicita una API Key tipo v3 auth). Se guarda solo en este navegador.
