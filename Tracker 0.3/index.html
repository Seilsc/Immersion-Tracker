Ajustes

Conecta APIs externas para calcular automáticamente la duración de vídeos, series y películas en las actividades de tipo "duración del contenido".

API de YouTube

Calcula la duración real de un vídeo a partir de su enlace. Consíguela gratis en Google Cloud Console: crea un proyecto → activa "YouTube Data API v3" → crea una credencial de tipo API key.

API de TMDB

Busca series, animes y películas para obtener su duración media por episodio o total. Consíguela gratis en themoviedb.org/settings/api (crea una cuenta → Configuración → API → solicita una "API Key" v3 auth).

Privacidad

Estas claves, y todas tus sesiones registradas, se guardan únicamente en el almacenamiento local de este navegador (localStorage). No se envían a ningún servidor de Anthropic ni de terceros, salvo a las propias APIs de YouTube y TMDB cuando las uses.
